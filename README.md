# Ejercicios de Listas en C# – Estructura de Datos

Este repositorio contiene 5 ejercicios básicos en C# enfocados en el uso de listas (List<T>), arreglos, bucles y entrada/salida por consola.
Los ejercicios están basados en problemas introductorios comunes de Estructura de Datos y sirven para reforzar la lógica de programación.

# Mostrar asignaturas de un curso

Descripción:
Almacena las asignaturas de un curso en una lista y las muestra por pantalla.

# Mostrar mensaje “Yo estudio <asignatura>”

Descripción:
Almacena las asignaturas en una lista y muestra el mensaje
“Yo estudio <asignatura>” para cada una.

# Notas por asignatura

Descripción:
El programa:

*Guarda las asignaturas en una lista
*Pide al usuario la nota de cada asignatura
*Muestra el resultado final por pantalla

# Números ganadores de la lotería

Descripción:
Solicita al usuario 6 números, los almacena en una lista, los ordena de menor a mayor y los muestra por pantalla.

# Mostrar números del 1 al 10 en orden inverso

Descripción:
Guarda los números del 1 al 10 en una lista y los muestra en orden inverso, separados por comas.
